9. 확인 불가 및 보완 우선순위

자동 판정 한계

  • 시연 영상의 실제 성공 여부는 Agent가 판정하지 않고, 타임스탬프 확인표를 평가자에게 전달한다.
  • R-01, R-03, R-04, R-13, R-17, R-20, R-22, R-26은 로그 또는 현장 조건 검증이 중요하다.
  • ZIP에는 git_log.txt가 포함되어 협업 분석 참고는 가능하지만, PR 리뷰 여부는 로컬 Git 로그만으로 확인이 제한된다.

보완 권장

  • 시연 영상의 실제 성공 여부는 최신 제공 타임스탬프 기준으로 평가자가 최종 확인한다.
  • 오프라인 LAN, 실제 카메라 FPS, TTS/LLM 폴백은 현장 장비 조건의 영향을 받으므로 첨부 로그와 시연 영상을 함께 확인한다.
